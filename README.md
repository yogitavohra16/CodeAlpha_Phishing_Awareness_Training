# Phishing Awareness Training Portal

## Overview

The Phishing Awareness Training Portal is an interactive cybersecurity education website developed as part of the CodeAlpha Cyber Security Internship Program.

The purpose of this project is to educate users about phishing attacks, social engineering techniques, fake websites, malicious emails, and online scams. The portal provides easy-to-understand explanations, real-world case studies, prevention techniques, and an interactive quiz to help users improve their cybersecurity awareness.

---

## Objectives

* Educate users about phishing attacks.
* Explain how attackers use social engineering techniques.
* Teach users how to identify phishing emails and fake websites.
* Provide practical cybersecurity safety tips.
* Improve user awareness through interactive learning and quizzes.

---

## Features

### Home Page

* Modern responsive user interface
* Cybersecurity-themed design
* Smooth navigation between sections

### What is Phishing?

* Detailed explanation of phishing attacks
* Real-life phishing examples
* Common phishing targets
* Safety recommendations

### Types of Phishing

* Email Phishing
* Spear Phishing
* Smishing (SMS Phishing)
* Vishing (Voice Phishing)

### Social Engineering Techniques

* Urgency and Fear Tactics
* Impersonation
* Authority Abuse
* Curiosity-Based Attacks
* Trust Exploitation

### Phishing Email Detection

Learn how to identify suspicious emails by checking:

* Sender address
* Grammar mistakes
* Suspicious links
* Urgent requests
* Unexpected attachments

### Fake Website Detection

Users learn to identify:

* Misspelled domains
* Missing HTTPS
* Fake login pages
* Suspicious URLs
* Poor website design

### Real-World Phishing Case Studies

Includes examples of:

* Google and Facebook phishing scams
* COVID-19 phishing campaigns
* Banking fraud attacks
* Credential theft incidents

### Prevention and Safety Tips

* Verify sender identity
* Use strong passwords
* Enable Multi-Factor Authentication (MFA)
* Avoid suspicious links
* Keep software updated

### Interactive Quiz

* 10 cybersecurity awareness questions
* Instant feedback
* Score calculation
* User-friendly learning experience
